# Datalab2 Samenwerkingsovereenkomst

# Afspraken
1. Deadlines worden afgesproken en gehaald.
2. Pauzes mogen op elk moment gehouden worden, maar maximaal 30 minuten.
3. GenAI, GenAI mag gebruikt worden voor het schrijven van kleine of repetetieve stukken code, maar deze moet altijd volledig begrepen worden en uitgelegd moeten kunnen worden.
4. Als iemand niet aanwezig kan zijn wordt dit minimaal 24 uur van tevoren gecommuniceerd. Met uitzondering van onbeziene omstandigheden.
5. Voor het niet halen van deadlines moet dit minimaal 48 uur van te voren worden gecommuniceerd.
6. Code commits worden altijd minimaal beschreven en zijn niet 'update' of andere termen
7. Code commits gebeuren altijd alleen in eigen branche, en main branche commits worden altijd in overleg gedaan. Zo voorkomen we merge conficts.

# Sancties
- Als iemand een afspraak breekt, moet deze volgende keer een tractatie meenemen voor de groep.
